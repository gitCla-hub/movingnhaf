about

✕

This project explores displacement as a state of drifting without ever fully arriving. Words entered by users become like living particles, scattering, breaking apart, and reconnecting in unexpected ways. Built with p5.js, the piece uses randomness, motion, and interactivity to keep the composition alive — particles fade, float, and form new connections in response to user input. Instead of fixed structures, it creates a shifting network that grows through interaction, inviting users to experience how identities, memories, and connections are always in motion, fragile yet continually reforming.
